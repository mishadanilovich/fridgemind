# Промпт-добавка для Claude Design — избранное, шаблоны меню, импорт рецептов

Отправь это следующим сообщением в тот же чат Claude Design — это правки экранов "Рецепты" и "Меню на неделю", не новые экраны.

---

Доработай следующие экраны в этом прототипе:

**1. Избранное для рецептов.** На карточке рецепта (в списке и на экране просмотра) добавь иконку "избранное" (например, сердечко или звёздочка) — по нажатию переключается сразу, без подтверждения. На экране "Рецепты" рядом с полем поиска добавь переключатель/чип "Только избранное" — сужает список рецептов до отмеченных.

**2. Импорт рецептов.** В шапке экрана "Рецепты" добавь кнопку/иконку "Импортировать". По нажатию — экран с двумя вариантами: **"Сгенерировать промпт для ИИ"** и **"У меня уже есть готовый JSON"**.

- Для "Сгенерировать промпт": текстовое поле "Опишите, какие рецепты нужны" (плейсхолдер-пример: "5 быстрых ужинов для семьи с ребёнком, без грибов, с курицей и рыбой") → кнопка "Сгенерировать" → показывает получившийся текстовый промпт целиком (в виде текстового блока, как в мессенджере) и кнопку **"Скопировать"** — пользователь сам вставляет его в любой ИИ-чат.
- Дальше (для обоих вариантов) — экран с большим текстовым полем "Вставьте JSON сюда" и отдельной кнопкой "или загрузить файл" рядом/под полем.
- После этого — экран подтверждения импорта: список рецептов, которые будут добавлены (название, число ингредиентов, число шагов — компактные карточки-превью), с пометкой у тех ингредиентов, что не нашлись в справочнике (нужен выбор категории или сопоставление вручную), и кнопками "Импортировать" / "Отмена" внизу.

**3. Шаблоны меню.** На экране "Меню на неделю" добавь действие "Сохранить как шаблон" (например, в меню из трёх точек в шапке недели) — по нажатию показывается простой диалог с полем ввода названия шаблона (например, "Обычная неделя") и кнопкой "Сохранить". Отдельно добавь действие "Применить шаблон" — открывает список ранее сохранённых шаблонов (карточки с названием), при выборе — предупреждение "Это заменит уже запланированные рецепты на этой неделе" с кнопками "Применить"/"Отмена".

Все элементы — в уже установленном стиле прототипа.

## После получения макета

Экспортируй обновлённые экраны в Claude Code, ориентируясь на CLAUDE.md: раздел 3 ("Спланировано детально (следующая итерация после MVP)"), раздел 5 ("Избранное для рецептов", "Шаблоны меню (план после MVP)", "Импорт рецептов из JSON-файла (план после MVP)").
