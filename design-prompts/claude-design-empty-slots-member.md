# Промпт-добавка для Claude Design — скрыть пустые слоты для роли "Участник"

Отправь это следующим сообщением в тот же чат Claude Design — это правка состояния экранов "Сегодня" и "Меню на неделю" для роли "Участник", не новый экран.

---

Доработай состояние роли "Участник" на экранах **"Сегодня"** и **"Меню на неделю"**: пустые слоты приёма пищи (те, в которые ещё не назначен рецепт) для этой роли должны **вообще не отображаться** — раз Участник всё равно не может назначить туда рецепт, пустая карточка "+ добавить рецепт" ему не нужна и только загромождает экран. Показывай только те слоты, где уже есть назначенный рецепт.

Для остальных ролей (Организатор, Редактор) пустые слоты по-прежнему видны как обычно, с возможностью назначить рецепт — это касается только вида для роли "Участник".
